# 🧠 DarkmoorAI

Intelligence, evolved.

DarkmoorAI is a powerful AI assistant platform that combines DeepSeek's advanced language models with free knowledge sources (Wikipedia, arXiv, PubMed, Open Library) to provide accurate, sourced answers with complete privacy.

## ✨ Features

- 💬 Chat with AI assistant
- 📁 Upload PDFs, Word docs, Excel files, images
- 🌐 Real-time Wikipedia integration (FREE)
- 🔬 Scientific paper search (arXiv, PubMed)
- 📚 Book database (Open Library, Gutenberg)
- 🔒 Privacy-first design
- 🧠 Pure DarkmoorAI branding
- 💰 Cost optimized (.05-0.15/user/month)

## 📦 Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS, Redux
- **Backend**: FastAPI, Python, PostgreSQL, Redis
- **AI/ML**: DeepSeek API, Sentence Transformers, FAISS
- **Free APIs**: Wikipedia, arXiv, PubMed, Open Library, Gutenberg
- **Infrastructure**: Docker, Kubernetes, Terraform
